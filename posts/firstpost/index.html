Getting setup with .Net Core, MonoGame and SadConsole

I did this on both MacOS and Linux and it was crazy easy.

Just follow the instructions #

There's a guide on the SadConsole website which explains what you need to do to create a new project. There isn't anything about the guide that's specifically for Windows, just follow the instructions if you're on MacOS or Linux.

Summary of installation instructions:

  1. Install .Net Core (Version 3.1 at the time I wrote this)
  2. Install MonoGame (3.7.1 when I wrote this)
  3. Creating a working folder and change directory into it
  4. Install the SadConsole.Templates (this will really work across operating systems)
  5. Create a project
  6. Run the project with dotnet run
